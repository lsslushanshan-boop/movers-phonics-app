# Movers Phonics Cards

这是一个手机优先的极简背单词网页 app。

## 功能

- 单次只展示 1 个剑桥少儿英语二级 `Movers` 单词
- 每个单词都按自然拼读规则拆解
- 每次进入单词卡自动播放一轮三段式朗读
- 朗读顺序是：整词 1 遍 -> 按自然拼读拆分读 1 遍 -> 整词再读 1 遍
- Movers 单词按生活场景顺序编排
- 点击 `下一个` 进入下一个单词
- 点击 `再读 3 遍` 可以重复播放

## 使用

直接在浏览器打开 `index.html`。

如果用手机浏览器打开，也可以把网页添加到主屏幕，当成简单 app 使用。

## 说明

- 朗读依赖浏览器的 `speechSynthesis`
- 当前词库为内置示例 Movers 常见词，后续可以继续扩充
